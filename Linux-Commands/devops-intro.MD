## DevOps Intro ##

What is DevOps?
```
Devops is a process of creating co-operation between development team and Operation team.
Dev means Devloper.
Ops means testing, Build and Release.
It follows the Agile methodology.
Devops works for both sofware and hardware
```
or
```
DevOps is a way of working where Software Developers(Dev) and Operations team(Ops) work together to build, test abd deliver software faster, better and more reliably.
```
